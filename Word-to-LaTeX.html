Word-to-LaTeX

Unofficial ConTeXt Wiki mirror

Last modified: 2020-06-08

Word-to-LaTeX is a nice MS Word to LaTeX converter that is very configurable. You can costumize the output file to have a format that is nearly 100% conTeXt, e.g. by teaching the program to write \starttext instead of \begin{document}. The program also can write a very clean output file, without all Word redundant font codes, etc. However, not every output command is configurable, and you'll probably find a few LaTeX commands (mainly in tables), but not a lot. You must read the file and correct some things before compiling.

The program can be downloaded from http://kebrt.webz.cz/programs/word-to-latex/. It only works with Windows.

The program can also make clean XML output from the Word file.

This is an example of a configuration file (you can make them through an easy GUI). With this one you'll get a nearly 100% conTeXt output with unicode encoding, no preamble and very clean. It's useful, for example, to convert Word articles that have to be incorporated as chapters in a book composed with TeX, etc. Save as context.xml or so.
