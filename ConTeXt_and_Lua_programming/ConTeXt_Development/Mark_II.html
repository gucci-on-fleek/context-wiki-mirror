ConTeXt Mark II

ConTeXt is under continuous development (See the FAQ) . At the time of writing (2020), the stable version is called Mark IV and the development version is called LMTX. Since around 2005, Mark IV has slowly replaced the earlier stable version that was called Mark II.

This wiki has been around long enough that there is quite some wiki content explicitly written for that now obsolete Mark II version of ConTeXt. And it is assumed that there are still users using that version as well.

This page is intended to list wiki pages with specific Mark II content, but see also the Category:Old Content Category.
