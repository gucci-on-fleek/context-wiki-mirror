Commands

Load collection of predefined colors

Many color collections are defined, which are not loaded by default. These can be loaded using \usecolor 

\usecolors[NAME]

where NAME is the name of the collection. The details of colors defined in these collections is shown below. You can also include such a detailed description of the colors defined in a color collection using \showcolor

Display collection of predefined colors 

\showcolor[NAME]

For example, the truncated output of \showcolor[solarized] is shown below. 

  • \showcolor[solarized]

Display components of defined colors

To view the components of a defined color, you can use \showcolorcomponents 

\showcolorcomponents[color-1, color-2]

For example 

  • \showcolorcomponents[red,middlered,darkred]
  • \colorcomponents
  • \transparencycomponents
  • \processcolorcomponents

For example, 

  • \startlines
\colorcomponents{red}
\transparencycomponents{red}
\processcolorcomponents{red}
\stoplines

Testing if color is defined

To test if a color is defined, use

  • \doifcolor
  • \doifcolorelse

Usage: 

\doifcolor{color-name}{... if branch ... }
\doifcolorelse{color-name}{... if branch ...}{... else branch ...}
