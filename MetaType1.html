MetaType1

Unofficial ConTeXt Wiki mirror

Last modified: 2025-01-12

MetaType1 is a MetaPost-based engine for generating Type 1 fonts, written by Bogusław Jackowski, Janusz M. Nowacki (and Piotr Strzelczyk ?). It was made to substitute Knuth's MetaFont, which is only capable of producing bitmap fonts.

(One can only use "fill" commands, pens are not supported.)

The utility was used to create:

  • Latin Modern fonts (a substitute for Knuth's Computer Modern)
  • Antykwa Toruńska, Antykwa Półtawskiego, Kurier & Iwona, see http://www.janusz.nowacki.strefa.pl/
