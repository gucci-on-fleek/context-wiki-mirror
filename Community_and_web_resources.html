Mailing lists

ConTeXt Mailing lists store years of precious Q&A. Please refer to the guidelines for writing to the mailing lists.

  • ntg-context@ntg.nl

    • General mailing list for ConTeXt users. Recommended for all ConTeXt users.
    • Current threads

  • dev-context@ntg.nl

    • Discussions between ConTeXt developers. Recommended for developers contributing macros and/or documentation.
    • Current threads

  • wiki@contextgarden.net

    • Discussions regarding the ConTeXt wiki. Recommended for all ConTeXt Wiki contributors.
    • Current threads
  • Search in the archives hosted by mail-archive.com :
    • ntg-context archives
    • dev-context archives
  • ConTeXt Mailing lists are hosted by:
    • the Dutch Language TeX User Group NTG - Mailman interface
    • the TeX Users Group TUG - Mailman interface
    • the ContextGarden - Mailman interface
  • Other TeX Engines specific mailing-list
    • luatex@tug.org for LuaTeX
    • xetex@tug.org for XeTeX

Chats, Mastodon, on-line meet-up

Magazines

  • ConTeXt Group Journal . Actual PDFs can be downloaded starting about six months after publication of each journal issue. Until then, there is only a Table of Contents available.
  • TUG Boat and TUGboat index , published three times a year.
  • MAPS, the NTG's journal and MAPS index . It has a lot of articles about ConTeXt and related subjects. You can search the articles .
  • This Way Official ConTeXt magazine
  • My Way ConTeXt magazine by its users
  • PracTeX Journal is (was ? OLD) an online journal with several articles about ConTeXt. Some articles (those related to ConTeXt) are archived .
