Command/contextversionnumber

Unofficial ConTeXt Wiki mirror

Last modified: 2020-08-29

Syntax

\contextversionnumber

Description

The command \contextversionnumber provides the name of the engine used.

Example 

  • \setuppapersize[A5]

This Wikipage was rendered with\ \Context\ version\ \contextversionnumber.\crlf
It uses \texenginename\ version \texengineversion\ as engine.

  • % mode=mkiv
\setuppapersize[A5]

This Wikipage was rendered with\ \Context\ version\ \contextversionnumber.\crlf
It uses \texenginename\ version \texengineversion\ as engine.

See also

  • \texenginename gives the name of the engine (e.g. LuaTeX, or XeTeX).
  • \texengineversion gives the version number of the engine (e.g. 0.64).
  • \contextversion gives the timestamp of the ConTeXt version in use
  • \contextmark gives MKII or MKIV, the ConTeXt major release marker
  • \contextversionnumber gives the ConTeXt version followed by the mark.
