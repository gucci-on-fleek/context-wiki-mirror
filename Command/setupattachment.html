Summary

The command \setupattachment is used for attachment settings.

Settings

\setupattachment[...,...][...=...,...]
[...,...] name
title text
subtitle text
author text
file file
name name
buffer buffer
type name
state start stop
method normal hidden
symbol Graph Paperclip Pushpin Default
width fit dimension
height fit dimension
depth fit dimension
color color
textlayer name
location leftedge rightedge inmargin leftmargin rightmargin text high none
distance dimension

Option Explanation
title
symbolic name of the attachment
subtitle
keywords
file
file name / path
name
output filename
buffer
name of a buffer (instead of an external file)
type
set the MIME type of an attachment, as needed for PDF/A
method
normal show
hidden don’t show
width
width of the symbol
height
height of the symbol
color
color of the symbol
textlayer
layer for the placement of the symbol
location
placement of the symbol

Description

This is for file attachments, i.e. files that get embedded in your PDF.
