Summary

The command \setupsynctex is used to configure SyncTeX support

Settings

\setupsynctex[...=...,...]
state start stop
method min max

Option Explanation
state
default: stop (don’t write .synctex file)

start: write .synctex file (in modern ConTeXt syntax)

repeat: write .synctex file in more traditional syntax
method
min save information for every word
max save information for bigger areas (more efficient)
