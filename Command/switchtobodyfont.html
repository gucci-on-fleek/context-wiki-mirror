Command/switchtobodyfont

Unofficial ConTeXt Wiki mirror

Last modified: 2025-05-01

Syntax (autogenerated)

\switchtobodyfont[...,...]
[...,...] inherits from \setupbodyfont

Syntax

\switchtobodyfont[...,...,...]
[...,...,...] 5pt ... 12pt small big global

Description

Use \setupbodyfont to change the font size for whole document, \switchtobodyfont doesn’t change the size for the header/footer.

Example 

  • \input ward

\switchtobodyfont[pagella,small]

\input ward
