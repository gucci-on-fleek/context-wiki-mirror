Summary

The command \setuppairedbox is used for setting up the connection between two paired boxes.

Settings

\setuppairedbox[...,...][...=...,...]
[...,...] name
before command
after command
inbetween command
location left middle right top bottom high low
n number
distance dimension
bodyfont see \setupbodyfont
width dimension
height dimension
maxwidth dimension
maxheight dimension
align see \setupalign
style style command
color color

Option Explanation
location
controls the placement of the second argument with respect to the first argument of the placepairedbox command.
