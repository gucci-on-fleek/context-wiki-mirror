Summary

The command \blackrule is drawing a rule.

Settings

\blackrule[...=...,...]
...=...,... inherits from \setupblackrules

Description

Draws a horizontal filled box, called a rule, with some configuration options.

Examples

Example 1 

  • \blackrule %default rule

\blackrule[color=red, height=1em] % customized rule
