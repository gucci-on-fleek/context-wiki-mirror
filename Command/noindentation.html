Summary

The command \noindentation disables indentation for a paragraph

Description

Disable indentation on one paragraph. (Not to be confused with the deprecated \noindenting , which has a more permanent effect: that command is equivalent to \setupindenting [no] )

Examples

Example 1 

  • minus one\par
zero\par
\setupindenting[yes, 20pt]
one\par
two\par
three\par
\noindentation four\par
five\par
