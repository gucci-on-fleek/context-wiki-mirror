Syntax

\referral[...,...=...,...]
bet text : concerns (NL: betreft)
ken text : mark (NL: kenmerk)
dat text : date (NL: datum)
van text : from (NL: van)
aan text : to (NL: aan)
ref text

Description

Construct a business letter's 'information block'.

Example 

  • \referral[
    van=Hans,
    aan=Leslie,
    dat=\currentdate,
    ken=\referraldate/HH/diplomacy,
    bet={Our respective macropackages},
    ref=Our mutual friend Donald
]
