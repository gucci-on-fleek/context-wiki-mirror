Summary

The environment \startsubstack ... \stopsubstack is used for stacked subscripts

Description

This environment allows placement of a list of subscripts as a tight stack. The individual items are separated by \NR .

Examples

Example 1 

  • \startformula
    \sum_{\startsubstack
        1 \le i \le p \NR
        1 \le j \le q \NR
        1 \le k \le r \NR
    \stopsubstack}
    a_{ij}b_{jk}c_{ki}
\stopformula
