Summary

The command \pagereference creates a page reference.

Settings

\pagereference[...,...]
[...,...] reference

Description

Examples

Example 1 

Look in \in{chapter}[hotel] on \at{page}[hotel] for a complete
overview of accommodation in \pagereference[accommodation]Hasselt.

A chapter number and a page number will be generated when processing the input file. On another spot in the document you can refer to accommodation with; 

\at{page}[accommodation].
