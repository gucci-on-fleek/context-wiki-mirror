Summary

The command \hl draws a short horizontal rule

Settings

\hl[...]
[...] number

Description

Draws a horizontal rule with a width of 1em, multiplied with the optional factor. The horizontal rule is drawn on top of the baseline.

Examples

Example 1 

  • \obeylines
\tex{hl} draws a horizontal line like this: \hl
\tex{hl[2]} draws a horizontal line like this: \hl[2]
