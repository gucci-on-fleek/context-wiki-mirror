Summary

The command \high creates a superscript in text mode

Settings

\high{...}
{...} content

Description

Create a superscript in text mode.

Examples

Example 1 

  • Today is the 3\high{rd} March 2010. \par
The lethal concentration L\low{50} varies with the temperature.\par
Alpha particles are \lohi[left]{2}{4}He\high{2+}. \par
