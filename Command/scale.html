Summary

The command \scale is used for scaling an object.

Settings

\scale[...][...=...,...]{...}
[...] name
...=...,... inherits from \setupscale
{...} content

Description

Scale content using different methods.

See \setupscale for parameter details.

Examples

Example 1 

  • \setuppapersize[A5]
Spam.                        \\
\scale[factor=fit]   {Spam.} \\
\scale[wfactor=max]  {Spam.} \\
More spam.                   \\
\scale[scale=1500]   {Spam.} \\
\scale[xscale=1500]  {Spam.} \\
\scale[lines=2]      {Egg.}  \\
\scale[sx=2]         {Egg.}  \\
\scale[grid=halfline]{Sausage.}  \\
