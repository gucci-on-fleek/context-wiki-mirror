Summary

The command \showfontstrip is used visualising the current typeface combination, with roman, sans, math, and teletype families, with x-heights and m widths.

Settings

\showfontstrip[...,...]
[...,...] inherits from \setupbodyfont

Description

Examples

Example 1 

  • \setupbodyfont[pagella,7pt]
\framed[frame=off]{\showfontstrip}
