Summary

The command \setuplayer is used to set the options for a layer.

Settings

\setuplayer[...,...][...=...,...]
[...,...] name
state start stop continue next repeat
option test
page number
width dimension
height dimension
region name
rotation left right inner outer number
position yes no overlay
hoffset max dimension
voffset max dimension
offset dimension
dx dimension
dy dimension
sx number
sy number
x dimension
y dimension
corner left middle right top bottom
location grid left middle right top bottom line l c r m t b g
line number
column number
direction normal reverse
doublesided yes no
method fit overlay
repeat yes
preset lefttop righttop leftbottom rightbottom middle middletop middlebottom middleleft middleright lefttopleft lefttopright name

Option Explanation
Name of the layer
state
start layer is visible
stop layer is invisible
repeat layer gets repeated on every page
page
layer is shown only on this page (makes more sense for \setlayer)
width
width of the layer (doesn’t limit elements)
height
height of the layer
region
placement relates to this named region (unclear, plase add a working example if you know)
hoffset
horizontal offset from reference point (see presets)
voffset
vertical offset from reference point (see presets)
x
horizontal position from left edge of paper
y
vertical position from top edge of paper
corner
corner of reference point (see presets)
line
placement of elements in a virtual column grid (see \setuplayout)
column
placement of elements in a virtual column grid (see \setuplayout)
repeat
repeat on all pages?
preset
placement preset
