Command/setupfootnotedefinition

Unofficial ConTeXt Wiki mirror

Last modified: 2021-10-29

Syntax (autogenerated)

Syntax

\setupfootnotedefinition[...,...=...,...]
[...,...=...,...] see \setupdescription

Description

Setup the style of the footnote (not of the in-text footnote mark).

Example 

  • \setuppapersize[A10]

\setupfootnotedefinition
   [headstyle=italic, 
    location=top, 
    color=red, 
    headcolor=blue]

Lorem\footnote{ipsum}

See also

  • \setupfootnotes to setup the blocks of multiple footnotes
  • \setupdescriptions , as footnotes are implemented as description objects.
