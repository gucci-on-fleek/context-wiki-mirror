Summary

The command \setupfittingpage is used for pages of varying size that fit tightly around their contents.

Settings

\setupfittingpage[...,...][...=...,...]
[...,...] name
pagestate start stop
command oneargument
margin page
scale number
paper auto default name
...=...,... inherits from \setupframed
