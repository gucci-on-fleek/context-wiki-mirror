Summary

The command \hphantom creates an invisible item that has width but no height.

Settings

\hphantom{...}
{...} content

Description

\hphantom typesets the supplied content and then throws it away, leaving a 'phantom' i.e. an invisible item that has zero height but the width of the phantom text.

Examples

Adding horizontal space in the middle of a sentence

(Something similar can be achieved with \hspace) 

  • I want some space between here \hphantom{mmmmm} and here.
