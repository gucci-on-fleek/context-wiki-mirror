Summary

The command \setwidthof is used to find out the width of a piece of text.

Settings

\setwidthof...\to\command
... text
\to delimiter
\command csname

Description

Examples

Example 1 

  • \setwidthof{ABCDE}\to\MyWidth
{ABCDE} is \MyWidth\ wide.

\framed[width=\MyWidth,]{ABCDE}
\framed[width=\MyWidth,]{ABC}
