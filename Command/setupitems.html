Summary

The command \setupitems is used to configure item lists (an alternative to enumerations).

Settings

\setupitems[...,...][...=...,...]
[...,...] name
alternative left right top bottom inmargin
textwidth dimension
width dimension
distance dimension
symbol none name
align left middle right flushleft flushright inner outer margin
textalign left middle right flushleft flushright inner outer
n number
before command
after command
inbetween command
