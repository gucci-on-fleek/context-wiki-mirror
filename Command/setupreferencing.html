Summary

The command \setupreferencing changes the output of the \about command.

Settings

\setupreferencing[...=...,...]
state start stop
expansion yes no xml
autofile yes no
export yes no
left command
right command
width dimension
doublesided yes no
interaction label text symbol all
separator command
prefix + - text
labelalternative inleftmargin inrightmargin inoutermargin ininnermargin inleftedge inrightedge inouteredge ininneredge atleftmargin atrightmargin inleft inright inouter ininner inmargin inother margintext name
labelstyle style command
labelcolor color

Description

left= and right= apply to the left and right of \about .
