Summary

The command \setupbuffer configures buffers.

Settings

\setupbuffer[...,...][...=...,...]
[...,...] buffer
before command
after command
strip yes no

Option Explanation
before
Command to execute before the contents of the buffer are typeset. Not used around \typebuffer.
after
Command to execute after the contents of the buffer are typeset. Not used around \typebuffer.
strip
yes Strip common leading spaces from all lines

Description

Configure a named buffer, or the default \startbuffer .
