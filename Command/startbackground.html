Summary

The environment \startbackground ... \stopbackground is used for backgrounds

Settings

\startbackground[...=...,...] ... \stopbackground
...=...,... inherits from \setupbackground

Description

This is the old way to do backgrounds behind text.

This mechanism is sort of obsolete , \starttextbackground should ideally be used instead.
