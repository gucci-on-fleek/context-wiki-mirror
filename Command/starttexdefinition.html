Summary

The environment \starttexdefinition ... \stoptexdefinition is an alternative to \define.

Examples

Example in extended mode

In MkVI/MkLX mode (activated by a first line as shown or a file extension of .mkvi or .mklx) you can use named parameters: 

% macros=mkvi (.mkvi)
\starttexdefinition WebToo #Output#URL
\goto{#Output}[url(#URL)]
\stoptexdefinition
