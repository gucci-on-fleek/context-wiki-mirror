Summary

The command \pagenumber returns the current page number (before conversion).

Description

Note that \pagenumber and \userpagenumber only work reliably when used in a header or footer. When used in running text they may be out by one if the page breaking algorithm decides to go back and break the page before the point where the page number value was inserted.
