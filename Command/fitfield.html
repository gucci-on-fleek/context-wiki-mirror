Summary

The command \fitfield is used to create interactive fields

Settings

\fitfield[...][...=...,...]
[...] name
...=...,... inherits from \setupfieldbody

Description

This command is like \fieldbody but with an extra defined \framed instance wrapped around it.

That extra framed is called fitfieldframed, and can be configured using \setupframed.
