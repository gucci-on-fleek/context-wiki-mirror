Summary

The command \defineframed creates a special instance of \framed .

Settings

\defineframed[...][...][...=...,...]
[...] name
[...] name
...=...,... inherits from \setupframed

Option Explanation
the new name
an optional parent framed instance

Description

Examples

Example 1 

  • \defineframed
  [GreyBg]
  [background=color,backgroundcolor=gray]
\GreyBg[frame=on]{Hello}
