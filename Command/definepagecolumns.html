Summary

The command \definepagecolumns is used to define specific page column sets

Settings

\definepagecolumns[...][...][...=...,...]
[...] name
[...] name
...=...,... inherits from \setuppagecolumns

Option Explanation
name the name that will be defined
name optional inheritance from existing named page columns
