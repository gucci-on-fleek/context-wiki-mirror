Summary

The command \setcharacterkerning is used for changing the spacing between characters.

Settings

\setcharacterkerning[...]
[...] reset name

Description

Examples

Example 1 

  • {Coucou}

{\setcharacterkerning[extrakerning]Coucou}

\definecharacterkerning [hugekerning] [factor=.250]

{\setcharacterkerning[hugekerning]Coucou}
