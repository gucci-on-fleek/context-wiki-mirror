Summary

The command \mathematics is used to typeset maths

Settings

\mathematics[...]{...}
[...] name
{...} content

Settings assignment

\mathematics[...=...,...]{...}
...=...,... inherits from \setupmathematics
{...} content

Description

Typeset the argument in mathematics mode

Examples

Example 1 

  • \mathematics{c = \sqrt{a^2 + b^2} }
