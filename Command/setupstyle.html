Summary

The command \setupstyle is usable to setup a font style.

Settings

\setupstyle[...,...][...=...,...]
[...,...] name
method command
style style command
color color

Option Explanation
method
\definestyle[MyStyle][method=command] also creates a command \MyStyle, otherwise you can use the new style only in style assignments.
style
e.g. bold, italic, bolditalic, slanted… or self defined values
color
color name
