Summary

The command \chem is used typesetting a chemical element.

Settings

\chem{...}{...}{...}
{...} text
{...} text
{...} text

Option Explanation
text element
text subscript
text superscript

Description

Write a chemical formula as element, subscript, superscript. The advantage over \lohi is that the \lohi command has to be placed between the element and its modifiers.

Examples

Example 1 

  • \chem{SO}{4}{2-} + \chem{H}{2}{}O
