Syntax (autogenerated)

Syntax

\CAP{...}
{...} text

Description

Obsolete in MkIV. In MkII, changes letters preceded by \\ to small capitals.

Example 

  • \contextversionnumber\crlf
\CAP{A \\user \\sometimes \\needs a SMAL\\L cap\\ital}
