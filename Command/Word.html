Summary

The command \Word capitalizes the first character of the text.

Settings

\Word{...}
{...} text

Description

Capitalizes the first character of the text.

Examples

Example 1 

  • \Word{on the origin of species}\crlf
\Word{'on the origin of species}

    yields
