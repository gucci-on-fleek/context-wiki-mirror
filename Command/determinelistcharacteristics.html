[...,...,...] name
[...,...=...,...] see \setuplist

Description

  1. list name (not combined list name!)
  2. see setuplist

Example

Place the Subject list only if the Subject list is not empty. 

\def\PlaceSubjectList%
  {\blank
   \determinelistcharacteristics[Subject]
   \doifmode{*list}{\placelist[Subject]}}
