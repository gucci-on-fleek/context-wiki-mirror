Syntax (automatically generated)

\definecombination[...][...][...=...,...]
[...] name
[...] name
...=...,... inherits from \setupcombination

Description

Use this command to define a named combination. Named combinations can inherit settings from other named combinations, or they can use explicit settings, or both, where the explicit settings then overrule the settings.
