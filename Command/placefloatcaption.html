Summary

The command \placefloatcaption is used to place a caption without a figure.

Settings

\placefloatcaption[...][...=...,...][...=...,...]
[...] name
title text
bookmark text
marking text
list text
reference reference
key value

Option Explanation
“userdata”, additional settings like subtitle. This makes only sense if you use your own caption setup.

Description

This makes sense if you place figures absolutely on layers or column spreads. This way you get the figure numbering without ConTeXt interfering in your image placement.
