Summary

The command \setlayertext is used to set (multiline) text on a layer

Settings

\setlayertext[...][...=...,...][...=...,...]{...}
[...] name
...=...,... inherits from \setuplayer
align see \setupalign
width dimension
style style command
color color
{...} text
