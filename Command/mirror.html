Summary

The command \mirror is used to mirror content

Settings

\mirror{...}
{...} content

Description

Mirror the content.

Examples

Example 1 

  • {\it asdf \mirror{asdf} }
\blank
\dontleavehmode
\externalfigure[cow][width=5.7mm]\mirror{\externalfigure[cow][width=6mm]}

Vertical mirror 

  • {\it \rlap{asdf}\mirror{\rotate[rotation=180]{asdf}}}
\blank
\dontleavehmode
\rlap{\externalfigure[cow][width=5.7mm]}%
\mirror{\rotate[rotation=180]{\externalfigure[cow][width=6mm]}}
