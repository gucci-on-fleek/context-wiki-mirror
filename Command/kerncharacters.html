Summary

The command \kerncharacters initiates character kerning (letterspacing).

Settings

\kerncharacters[...]{...}
[...] number
{...} text

Option Explanation
number implied unit of em

Description

This is an alias of \letterspacing and an instance of characterkerning (see \definecharacterkerning ).

The default value is 0.125

Examples

Example 1 

  • \kerncharacters[0.125]{HERE IS SOME KERNED TEXT} back to normal

Declarative version 

  • \start
\kerncharacters[0.125]
HERE IS SOME KERNED TEXT
\stop
back to normal
