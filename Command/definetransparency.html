Summary

The command \definetransparency is used to define a named transparency setting, independent of colors.

Settings 1

\definetransparency[...][...]
[...] name
[...] number

Settings 2

\definetransparency[...][...=...,...]
[...] name
a number none normal multiply screen overlay softlight hardlight colordodge colorburn darken lighten difference exclusion hue saturation color luminosity
t number

Settings 3

\definetransparency[...][...]
[...] name
