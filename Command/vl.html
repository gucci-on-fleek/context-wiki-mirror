Summary

The command \vl draws a short vertical rule

Settings

\vl[...]
[...] number

Description

The command draws a vertical rule with strut dimensions, multiplied by the factor specified in the optional argument. The height and depth are clipped to the baselinedistance, so that interline spacing is not influenced.

Examples

Example 1 

  • \tex{vl} draws a vertical rule \vl\ of a strut size, multiplied by\crlf
the factor in optional argument, as in \tex{vl}[3] \vl[3].
