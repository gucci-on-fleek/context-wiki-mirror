Summary

The command \setupcolors is used for general color settings.

Settings

\setupcolors[...=...,...]
state start stop
spot yes no
expansion yes no
factor yes no
rgb yes no
cmyk yes no
conversion yes no always
pagecolormodel auto none name
textcolor color
intent overprint knockout none

Option Explanation
state
Switch colors on or off globally.
stop All colors get converted to greyscale.
spot
Allow spot colors (for printing with more than CMYK, also for varnish)
no Defined spot colors get converted to RGB or CYMK
rgb
Allow RGB colors
cmyk
Allow CMYK colors
pagecolormodel
auto Prevent some issues with transparency
textcolor
Global main color for text
intent
This sets the complete document (global) to overprint (or knockout) and is only useful for grayscale-only documents.

Don’t set this parameter if you want to use local overprinting (or knockout)!

This setting is far older than the PDF/X or PDF/A related output intent parameter in \setupbackend, and there’s no direct relation other than that both parameters are prepress options.
