Summary

The command \setupbackend is used for standard compliance and color management settings

Settings

\setupbackend[...=...,...]

Option Explanation
export
yes create XML/HTML file(s) of the content, see Export
intent
output intent, i.e. printing conditions, necessary for PDF/X, see colorprofiles.xml
Coated␣FOGRA39␣(ISO␣12647-2:2004) Euroscale, offset, coated paper
ISO␣Coated␣v2␣300%␣(ECI) Euroscale, offset, coated paper
ISO␣Coated␣v2␣(ECI) Euroscale, offset, coated paper
ISOnewspaper26v4 Euroscale, web offset, newspaper paper
ISOnewspaper26v4_gr Grayscale, web offset, newspaper paper
ISO␣Uncoated Euroscale, offset, uncoated paper
ISO␣Uncoated␣Yellowish Euroscale, offset, yellowish uncoated paper
ISO␣Web␣Coated Euroscale, web offset, coated paper
Japan␣Color␣2001␣Coated Japan, coated paper
Japan␣Color␣2002␣Newspaper Japan, newspaper paper
Japan␣Web␣Coated␣(Ad) Japan, web offset, coated paper (advertisements)
SWOP2006_Coated3v2.icc SWOP (USA), offset, coated paper
SWOP2006_Coated5v2.icc SWOP (USA), offset, coated paper
Uncoated␣FOGRA29␣(ISO␣12647-2:2004) Euroscale, offset, uncoated paper
Web␣Coated␣FOGRA28␣(ISO␣12647-2:2004) Euroscale, web offset, coated paper
PSR_LWC_PLUS_V2_PT PSR, light weight coated paper
PSR_LWC_STD_V2_PT PSR, light weight coated paper
PSR_SC_PLUS_V2_PT PSR, coated paper
PSR_SC_STD_V2_PT PSR, coated paper
PSRgravureMF PSR, gravure printing
xmpfile
file include XMP metadata from this file
format
PDF standard, see PDF/X and PDF/A.
PDF/X-1a:2001 CMYK printing, CMYK images, no transparency, PDF 1.3
PDF/X-1a:2003 CMYK printing, CMYK images, no transparency, PDF 1.4
PDF/X-3:2002 CMYK printing, profiled images, no transparency, PDF 1.3
PDF/X-3:2003 CMYK printing, profiled images, no transparency, PDF 1.4
PDF/X-4 CMYK printing, profiled images, may contain transparency, profiles included, PDF 1.6
PDF/X-4p CMYK printing, profiled images, may contain transparency, profiles referenced, PDF 1.6
PDF/A-1a:2005 archiving
PDF/A-1b:2005 archiving
PDF/A-2a archiving
PDF/A-3a archiving
PDF/UA-1 accessibility
level
PDF compress level (values of 0..9 are allowed). Normally no need to change the default value.
option
always force embedding of the output intent profile (useful only in case of PDF/X-1a)
never don’t embed the output intent profile; no useful practise
profile
ICC based default color space profile (option for color management); this setting is document wide and currently only one profile per color space is allowed, it’s a list of {CMYK profile, RGB profile, Grayscale profile}
