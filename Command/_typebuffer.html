Summary

The instances of \typebuffer are used for verbatim display of named buffer contents.

Settings instance

\typeBUFFER[...=...,...]
...=...,... inherits from \setuptyping

Description

Fetches the content of a buffer class defined by \definebuffer to be typeset verbatim.
