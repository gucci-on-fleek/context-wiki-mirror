Summary

The command \definebar is used for defining text decorations like overstrikes and underbars.

Settings

\definebar[...][...][...=...,...]
[...] name
[...] name
...=...,... inherits from \setupbar

Option Explanation
new instance name
optional inheritance parent

Description

Creates a text decoration command. See bar for predefined instances as well as a set of examples.
