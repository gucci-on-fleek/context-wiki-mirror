Summary

The command \note places a footnote symbol

Settings

\note[...][...]
[...] name
[...] reference

Description

Places the symbol for the referenced footnote in the text. It does not create a footnote.

Examples

Example 1 

  • \setuppapersize[A7,landscape]
\starttext
\setupwhitespace[big]

---The noble sport of foot-to-ball!\footnote[fn:fball]{Association Football}

---Yes, indeed, the noble sport of soccer!\note[fn:fball]

(See footnote.\high{\tfxx \in[fn:fball]})
\stoptext
