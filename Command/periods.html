Summary

The command \periods is used to typeset a number of periods.

Settings

\periods[...]
[...] number

Settings assignment

\periods[...=...,...]
...=...,... inherits from \setupperiods

Description

With \periods one can place a sequence of periods. The number of periods to be placed is added in brackets. The periods have spaces between them, but not on the edges unless you add them.

Examples

Example 1 

  •    Two \periods[2] and four\periods[4]and ten \periods[10]
