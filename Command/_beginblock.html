Summary

The instances of \beginblock ... \endblock are used for text blocks that can be hidden or reused.

Settings example

\beginBLOCK[...,...][...=...,...] ... \endBLOCK
[...,...] + - name
key value

Description

There are no predefined instances. See \defineblock for further documentation.
