Summary

The command \bigm is used for math, to increase the size of delimiters characters used in the middle of expression

Settings

\bigm ...
... character

Description

Several commands are available for different sizes:

  • Size level 1 : \big*
  • Size level 2 : \Big*
  • Size level 3 : \bigg*
  • Size level 4 : \Bigg*

Examples

Example 1 

  • \setuppapersize[A7]
\setupbodyfont[16pt]
\starttext
$\Biggl{\{} \biggl{\{} \Bigl{\{} \bigl{\{} {\{} {\}} \bigr{\}} \Bigr{\}} \biggr{\}} \Biggr{\}}$

\blank[small]

$\Bigg {\{} \bigg {\{} \Big {\{} \big {\{} {\{} {\}} \big {\}} \Big {\}} \bigg{\}}  \Bigg {\}}$

\blank[small]

$\Biggm{\{} \biggm{\{} \Bigm{\{} \bigm{\{} {\{} {\}} \bigm{\}} \Bigm{\}} \biggm{\}} \Biggm{\}}$
\stoptext
