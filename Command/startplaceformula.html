Summary

The environment \startplaceformula ... \stopplaceformula is used to place a formula as a floating object, optionally numbered and captioned.

Settings

\startplaceformula[...=...,...] ... \stopplaceformula
title text
suffix text
reference + - reference

Option Explanation
title
caption
reference
reference anchor

Settings argument

\startplaceformula[...,...] ... \stopplaceformula
[...,...] + - reference

Description

Within \startplaceformula ... \stopplaceformula the display math block, \startformula ... \stopformula ist still necessary.
