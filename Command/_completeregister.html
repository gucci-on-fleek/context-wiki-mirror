Summary

The instances of \complete register are used to typeset a register.

Instances

\completeindex

Settings instance

\completeREGISTER[...=...,...]
...=...,... inherits from \setupregister

Description

The instances of \complete register are used to typeset a register, including a header section

Examples

Example 1 

  • \setuppapersize[A7,landscape][A6] \showframe
\setuppaper[mx=1,ny=2]
\setuparranging[XY]


\defineregister
  [author]

\starttext
Some text
\author{Jules Verne}
\author{HP Lovecraft}

\completeauthor % \completeregister[author]
\stoptext
