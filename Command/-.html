Syntax (autogenerated)

Syntax

\-

Description

Indicate an optional hyphenation point in a word.

Example 

  • \setuppapersize[A10]
\setupwhitespace[big]
supercalifragilisticexpialidocious

super\-cali\-fragi\-lis\-tic\-ex\-piali\-do\-cious
