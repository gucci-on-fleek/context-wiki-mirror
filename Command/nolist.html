Command/nolist

Unofficial ConTeXt Wiki mirror

Last modified: 2023-03-30

Syntax

\nolist{...}
{...} text

Description

An obsolete command, not present in the current source code. When used inside a sectioning command, the argument would appear in the title, but not in the list of contents. As mentioned in the mailing list (December 28, 2015) this command no longer exists.
