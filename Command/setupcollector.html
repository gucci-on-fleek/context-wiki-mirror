Summary

The command \setupcollector is used to define settings of a collector (like setuplayer)

Settings

\setupcollector[...,...][...=...,...]
[...,...] name
rotation left right inner outer number
corner left middle right top bottom
x dimension
y dimension
hoffset dimension
voffset dimension
location left middle right top bottom line l c r m t b g
state start stop
