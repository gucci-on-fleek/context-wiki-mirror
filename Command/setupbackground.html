Description

This mechanism is sort of obsolete and \setupbackgrounds (with an s) should be used instead.

Examples 

\setupbackground
    [frame=on]

\setupbackground
    [background=color, backgroundcolor=blue]

Add “location=paragraph” to \setuptextbackground, to remove the empty line after the itemize environment add “nowhite” to \startitemize.
