Summary

The instances of \startlabeltext ... \stoplabeltext are the environment version of labeltext

Instances

\startheadtext ... \stopheadtext
\startlabeltext ... \stoplabeltext
\startmathlabeltext ... \stopmathlabeltext
\starttaglabeltext ... \stoptaglabeltext
\startunittext ... \stopunittext
\startoperatortext ... \stopoperatortext
\startprefixtext ... \stopprefixtext
\startsuffixtext ... \stopsuffixtext
\startbtxlabeltext ... \stopbtxlabeltext

Settings instance

\startLABELtext[...][...] ... \stopLABELtext
[...] language
[...] key
