Summary

The command \ruby typesets CJK text including ruby.

Settings

\ruby[...]{...}{...}
[...] name
{...} text
{...} text

Option Explanation
optional named ruby (those are defined via \defineruby)
main text
ruby (script) text

Description

The command \ruby is used for CJK ruby text, like putting furigana over kanji characters.
