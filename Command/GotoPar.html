Summary

The command \GotoPar is used for ignoring the next blank line or \par. You have to add it at the end paragraph and not at the start.

Description

In the old MkII days \GotoPar (the commands is still available) did the job

With Luametatex you can now use the \ignorepars primitive to achieve the result.

Examples

Example 1 

  • This is some text.

This is some other text, it will appear in a new paragraph.

\blank

This is some text. \ignorepars % add space before \ignorepars

This is some other text, it will appear in a new paragraph.

\blank

This is some text. \GotoPar

This is some other text, it will appear in a new paragraph.

stop
