Syntax

\character{...}
{...} number

Description

Convert a number 1-26 to an uppercase letter.

Example 

  • % convert a number 1-26 to a lowercase letter
\character{3} -- \character{30}

% convert a number 1-26 to an uppercase letter
\Character{3} -- \Character{30}

% convert any number to lowercase letters (x, y, z, aa, ab, ac, ...)
\characters{3} -- \characters{30}

% convert any number to uppercase letters (X, Y, Z, AA, AB, AC, ...)
\Characters{3} -- \Characters{30}
