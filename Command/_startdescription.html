Summary

The instances of \startdescription ... \stopdescription are defined by \definedescription

Settings example

\startDESCRIPTION[...=...,...] ... \stopDESCRIPTION
title text
bookmark text
list text
reference reference

Settings example:title

\startDESCRIPTION[...,...]{...} ... \stopDESCRIPTION
[...,...] reference
{...} text
