Summary

The command \setupframedtext is used for configuring \startframedtext

Settings

\setupframedtext[...,...][...=...,...]
[...,...] name
left command
right command
inner command
location left right middle none
depthcorrection on off
bodyfont see \setupbodyfont
indenting see \setupindenting
style style command
color color
...=...,... inherits from \setupframed
...=...,... inherits from \setupplacement
