Summary

The instances of \see register are used for referring to another register entry.

Instances

\seeindex

Settings instance

\seeREGISTER[...]{...+...+...,...}{...}
[...] text processor->text
{...+...+...,...} index
{...} text processor->text

Option Explanation
sort entry
{...+...+...,...} index entry
reference entry

Description

The command 
\seeindex{rabbit}{rodent}

results in an entry in the index which looks like: 

rabbit
  see rodent


If you don’t like the “see”, you can change it using \setuplabeltext: 

\setuplabeltext[en][see={\emph{look up }}]
