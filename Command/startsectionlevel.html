Summary

The environment \startsectionlevel ... \stopsectionlevel is used for a “hierarchy neutral” structuring of texts.

Settings

\startsectionlevel[...][...=...,...][...=...,...] ... \stopsectionlevel
[...] name
title text
bookmark text
marking text
list text
reference reference
ownnumber text
placeholder text
key value

Description

This is a general replacement for structuring commands like \startchapter , \startsection etc. More information at \definesectionlevels .
