Syntax (autogenerated)

Syntax

{\frak...}
... text

Description

A switch: start using the Fraktur variant of the current font.

Example 

  • {\frak The quick brown fox jumps over the lazy dog. 1234567890}

See also

  • \goth for the Gothic font variant. (Which is seldom different.)
