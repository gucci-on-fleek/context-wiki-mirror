Syntax

\sub[ref,ref,...]
[ref,ref,...]

Description

Used in itemizations. Prints the item mark of the previous \item, with a + in front of it to indicate a subitem/continuation.

Example 

  • \startitemize[n]
    \item First item
    \sub continued
    \sub further continued
    \item Second item
\stopitemize
