Summary

The command \blackrules typesets a series of rules

Settings

\blackrules[...=...,...]
...=...,... inherits from \setupblackrules

Description

Typesets a series of rules with some configuration options.

Examples

Example 1 

  • \starttext
\blackrules
\stoptext

    produces

Example 2 

  • \setupblackrules[width=3cm]
\blackrules[alternative=a]\par
\blackrules[alternative=b]

    produces
