Command/keyword:contrastcolor

Unofficial ConTeXt Wiki mirror

Last modified: 2012-12-11

"ConTeXt distinguishes between references to the same page and to references which destination is on other pages. References which destination is on the same page don't obey the color setting. Instead the colour is controlled by a key called contrastcolor, which is red by default." — Marco, "Change color of links created with \in, \at, etc" on Stack Exchange.
