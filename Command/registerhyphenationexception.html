Command/registerhyphenationexception

Unofficial ConTeXt Wiki mirror

Last modified: 2025-09-14

Syntax

\registerhyphenationexception[...][...]
[...] language
[...] text

Description

With this command you can register a hyphenation exception. It takes an optional language code and list of words with hyphenation marks.

Example 

  • \registerhyphenationexception[Nietz-sche]
