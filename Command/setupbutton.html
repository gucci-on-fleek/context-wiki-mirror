Summary

The command \setupbutton is used to configure interactive buttons

Settings

\setupbutton[...,...][...=...,...]
[...,...] name
state start stop
samepage yes no empty none normal default
style style command
color color
contrastcolor color
alternative hidden
...=...,... inherits from \setupframed

Description

The command configures the style of buttons in interactive documents.
