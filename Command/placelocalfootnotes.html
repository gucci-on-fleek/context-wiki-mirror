Summary

The command \placelocalfootnotes is used to place local footnotes

Settings

\placelocalfootnotes[...=...,...]
...=...,... inherits from \setupnote

Description

Footnotes inside a \startlocalfootnotes environment are not numbered with the rest of the footnotes, and not placed automatically. Instead, they must be placed with \placelocalfootnotes
