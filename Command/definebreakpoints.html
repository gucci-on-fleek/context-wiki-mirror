Syntax (autogenerated)

\definebreakpoints[...]
[...] name

Syntax

\definebreakpoints[...]
[...] breakpoint set name

Description

\definebreakpoints[mycollection] initialises the set of breakpoints mycollection, which one can then add to with the \definebreakpoint command.

Example 

\definebreakpoint[compound]
