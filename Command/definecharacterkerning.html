Summary

The command \definecharacterkerning defines a named set of character kerning settings

Settings

\definecharacterkerning[...][...][...=...,...]
[...] name
[...] name
...=...,... inherits from \setupcharacterkerning

Description

There are two predefined settings: \kerncharacters and \letterspacing , both with the default value of 0.125.
