Syntax (autogenerated)

\getvariable{...}{...}
{...} name
{...} key

Description

Retrieve the value of the named variable in the specified namespace.

Example 

\setvariable{myvars}{uno}{1}

\getvariable{myvars}{uno} % result: 1
