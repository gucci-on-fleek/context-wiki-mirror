Summary

The command \usecolors loads a file with color definitions

Settings

\usecolors[...]
[...] file

Description

Load a file with color definitions. These are the MkIV color definitions:
Command Definition file Description
\usecolors[crayola] colo-imp-crayola.mkiv Crayola colors
\usecolors[dem] colo-imp-dem.mkiv Demo palettes and groups
\usecolors[ema] colo-imp-ema.mkiv Emacs colors
\usecolors[rainbow] colo-imp-rainbow.mkiv 'Rainbow' palette sets (same greyscale value for different hue sets)
\usecolors[ral] colo-imp-ral.mkiv RAL colors
\usecolors[rgb] colo-imp-rgb.mkiv RGB colors
\usecolors[solarized] colo-imp-solarized.mkiv Solarized colors ( http://ethanschoonover.com/solarized )
\usecolors[svg] colo-imp-x11.mkiv SVG colors
\usecolors[x11] colo-imp-x11.mkiv X11 colors
\usecolors[xwi] colo-imp-xwi.mkiv X Windows colors

Examples

Example 1 

  • \usecolors[dem] % use colors in colo-imp-dem.mkiv
