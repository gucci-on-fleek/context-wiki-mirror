Summary

The command \thinrules prints rules

Settings

\thinrules[...=...,...]
...=...,... inherits from \setupthinrules

Description

Prints rules that can be set up directly, or with \setupthinrules .

Beware: some options only apply when the command is used in vertical mode.

Examples

Example 1 

  • \setupthinrules[n=3]

test test \thinrules\ test test \par
test test \thinrules [color=green] test test \par
test test \thinrules [height=max, depth=max] test test \par

\thinrules
