Command/nofTABLErows

Unofficial ConTeXt Wiki mirror

Last modified: 2020-08-29

Syntax (autogenerated)

Syntax

\nofTABLErows

Description

Returns the number of rows the TABLE has.

Example 

\bTABLE
     \bTR
         \bTD (\currentTABLErow,\currentTABLEcolumn,\nofTABLErows,\nofTABLEcolumns) \eTD
         \bTD (\currentTABLErow,\currentTABLEcolumn,\nofTABLErows,\nofTABLEcolumns) \eTD
         \bTD (\currentTABLErow,\currentTABLEcolumn,\nofTABLErows,\nofTABLEcolumns) \eTD
         \bTD (\currentTABLErow,\currentTABLEcolumn,\nofTABLErows,\nofTABLEcolumns) \eTD
     \eTR
     \bTR
         \bTD (\currentTABLErow,\currentTABLEcolumn,\nofTABLErows,\nofTABLEcolumns) \eTD
         \bTD (\currentTABLErow,\currentTABLEcolumn,\nofTABLErows,\nofTABLEcolumns) \eTD
         \bTD (\currentTABLErow,\currentTABLEcolumn,\nofTABLErows,\nofTABLEcolumns) \eTD
         \bTD (\currentTABLErow,\currentTABLEcolumn,\nofTABLErows,\nofTABLEcolumns) \eTD
     \eTR
 \eTABLE
