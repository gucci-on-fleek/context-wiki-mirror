Summary

The instances of \presetlabeltext are used for providing default label values.

Instances

\presetheadtext
\presetlabeltext
\presetmathlabeltext
\presettaglabeltext
\presetunittext
\presetoperatortext
\presetprefixtext
\presetsuffixtext
\presetbtxlabeltext

Settings instance

\presetLABELtext[...][...=...,...]
[...] language
key value

Description

Not all labels exist in all languages. This command allows to set a default value that will not override any actually existing label text.
