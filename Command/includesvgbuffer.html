Summary

The command \includesvgbuffer is used for processing buffered svg

Settings

\includesvgbuffer[...][...=...,...]
[...] buffer
offset dimension

Option Explanation
name of the buffer
adds some offset

Description

The SVG is processed by ConTeXt via its internal XML processor and converted to Metapost graphic internally that is then included.
