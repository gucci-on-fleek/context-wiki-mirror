Summary

The instances of \copylabeltext copy label text groups

Instances

\copyheadtext
\copylabeltext
\copymathlabeltext
\copytaglabeltext
\copyunittext
\copyoperatortext
\copyprefixtext
\copysuffixtext
\copybtxlabeltext

Settings instance

\copyLABELtext[...][...=...,...]
[...] language
key key

Option Explanation
cd:key
the target label group
key the parent label group

Description

This command is used internally by the \definefloat command.

Examples

Example 1 

\copylabeltext[myfigure=figure]
