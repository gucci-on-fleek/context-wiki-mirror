Syntax

\placebookmarks[...,...,...][...,...,...]
[...,...,...] name
[...,...,...] name all

Description

Creates bookmarks for the headers "section": 

\placebookmarks[section]

Will do the same but open only chapter ones by default: 

\placebookmarks [chapter,section,subsection]
                [chapter]
