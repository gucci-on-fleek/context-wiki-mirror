Syntax

\signalrightpage

Description

Start tracking whether we are on a right- or a left-hand page. \signalrightpage is only necessary for text in the document. In the header it is sufficient to use \doifoddpageelse to set different values for left/right pages.

Example 

\signalrightpage\doifrightpageelse{Right!}{Left!}
