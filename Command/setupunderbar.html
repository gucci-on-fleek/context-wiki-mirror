Syntax

\setupunderbar[...,...=...,...]
alternative a b c
rulethickness dimension
bottomoffset dimension
topoffset dimension
rulecolor name

Description

  • alternative=a solid underline
  • alternative=b dashed underline
  • alternative=c dotted underline
