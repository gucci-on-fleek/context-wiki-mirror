Summary

The instances of \startformulaformula ... \stopformulaformula are used for specific displayed mathematical content

Instances

\startspformula ... \stopspformula spacebefore=none,spaceafter=none,indentnext=no,alternative=single
\startsdformula ... \stopsdformula spacebefore=none,spaceafter=none,indentnext=yes,alternative=single
\startmpformula ... \stopmpformula indentnext=no,alternative=multi
\startmdformula ... \stopmdformula indentnext=yes,alternative=multi

Settings instance

Description

Predefined cases of \startformula , with specific settings. The used arguments to \defineformula are given above.
