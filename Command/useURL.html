Description

text1 = URL;

file = attached file URL;

text2 = text.

comment

Do not forget:

  • \setupinteraction [state=start]

also useful;

  • \setupcolors [state=start]
  • \setupurl [color=blue]

Example 

  • \setupinteraction [state=start]

\setupcolors      [state=start]
\setupurl         [color=blue]

\setupinteraction [state=start,color=blue]
