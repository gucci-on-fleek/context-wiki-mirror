Command/usetypescript

Unofficial ConTeXt Wiki mirror

Last modified: 2025-01-18

Syntax (automatically generated)

\usetypescript[...][...][...]
[...] serif sans mono math handwriting calligraphy casual all
[...] name fallback default dtp simple
[...] size name

Description

Use this command to retrieve an earlier set of definitions made with the \starttypescript environment.

Internally called by \definetypeface

Example 

\usetypescript [palatino]
