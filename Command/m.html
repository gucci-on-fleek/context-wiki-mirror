Summary

The command \m puts its argument in inline math mode.

Settings

\m[...]{...}
[...] name
{...} content

Settings assignment

\m[...=...,...]{...}
...=...,... inherits from \setupmathematics
{...} content

Description

\m is the ConTeXt replacement for plain TeX’s $ ... $, but it takes additional options.

Examples

Example 1 

  • \tfd\m{e = m c^2}
