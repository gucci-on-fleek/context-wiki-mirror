Local document settings

Unofficial ConTeXt Wiki mirror

Last modified: 2024-09-17

ConTeXt will automatically load the file cont-loc.mkiv (and also cont-loc.mkxl when using LMTX) at the start of every run. This file isn't provided in any distributions, so it won't be overwritten by any updates; however, this means that you must create the file yourself.

To avoid issues where files work on one system but not others, it's probably best to limit the content of these files to commands like \enabledirectives and \enableexperiments.
