Configuration

Lsp server

The following screenshots show the use of digestif lsp-server.

In .config/kate/lspclient/settings.json 

{
    "servers": {
        "context": {
            "command": ["digestif"],
            "url": "https://github.com/astoff/digestif",
            "highlightingModeRegex": "^ConTeXt$"
        }
    }
}
