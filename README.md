ConTeXt Garden Wiki Mirror
==========================

This repository contains a mirror of the [ConTeXt Garden
Wiki](https://wiki.contextgarden.net/Main_Page). The mirror is generated
using a custom Python script that downloads and processes the wiki pages
and their associated resources.

This mirroring project [is hosted on
GitHub](https://github.com/gucci-on-fleek/context-wiki-mirror); the
(default) [`mirror`
branch](https://github.com/gucci-on-fleek/context-wiki-mirror/tree/mirror)
contains the mirrored content, while the [`master`
branch](https://github.com/gucci-on-fleek/context-wiki-mirror/tree/master)
contains the mirroring script and related files.
